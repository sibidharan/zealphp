# ZealPHP Documentation

Welcome to the official documentation for **ZealPHP** – the coroutine-powered PHP framework that modernises traditional LAMP applications while retaining the things you already love about the PHP ecosystem.

This documentation is broken up into small, focused guides.  If you are in a rush, read the first three pages and you will already be productive.

* [01 – Installation & First Run](01-installation.md)
* [02 – Routing Fundamentals](02-routing.md)
* [03 – Superglobals & the `G` helper](03-superglobals.md)
* [04 – Middleware](04-middleware.md)
* [05 – Coroutines, `go()` and `coproc()`](05-coroutines.md)
* [06 – Implicit routing (API & public folder)](06-implicit-routes.md)
* [07 – File-based API routing](07-api-routing.md)
* [08 – Porting an Apache/FPM project](08-porting-from-apache.md)

The documentation lives inside your project (`doc/` folder) so that it is always available – even when you are offline.

Happy coding with ZealPHP ❤️
